## FreeBusy
FreeBusy er en mobilapplikasjon som har som mål å forenkle koordinering av sosiale møter i grupper. Applikasjonen hjelper brukere med å finne felles ledige tidspunkter ved å sammenligne tilgjengelighet på gruppenivå, og reduserer behovet for manuell planlegging og frem-og-tilbake-kommunikasjon i meldinger.

Prosjektet er utviklet som en del av et studieprosjekt og befinner seg på et MVP-nivå. Fokus har vært på å validere kjerneidéen: å gjøre det raskere og enklere for grupper å bli enige om et tidspunkt for å møtes, ved hjelp av kalenderintegrasjon og en strukturert brukerflyt.

## Teknologi
FreeBusy er utviklet med følgende teknologier:
	•	React Native med Expo for kryssplattform mobilutvikling
	•	Firebase Authentication for brukerinnlogging og autentisering
	•	Firebase Firestore for lagring av brukere, grupper og avtaler
	•	Firebase Cloud Functions / Cloud Run for backend-logikk
	•	Google Calendar API for henting av kalenderdata og tilgjengelighet

## Slik kjører du prosjektet lokalt

1. Last ned prosjektet
- Last ned repositoriet som ZIP-fil
- Åpne prosjektet i Visual Studio Code

2. Installer nødvendige pakker:
- Kjør npm install for å installere nødvendige pakker.

3. Start applikasjonen:
Kjør npx expo start for å starte applikasjonen.

4. Test applikasjonen på en fysisk enhet:
- Last ned Expo Go-appen på din iOS- eller Android-enhet.
- Skann QR-koden som vises i terminalen.
